GlobalMeet Mobile

🌎 PGi

GlobalMeet Mobile is a platform for web conferencing and unified business communications. I led UX for these iOS and Android apps from 2018 to 2020, focusing on UI updates and new features like smart notifications, messaging, calling and video conferencing.

👈 Back Connect on LinkedIn 🔗

🔬 User and Usability Research

I conducted usability tests, interviews and surveys to shed empirical light on tough design problems. This research helped us understand user priorities and how important tasks were accomplished in the app.

🌧 Before

The original app only provided basic web conferencing features. It was inconsistent with other platforms, lacked a cohesive UI system, and looked dated.

🌈 After

I gave the mobile apps a complete overhaul using a consistent cross-platform UI system. A bevy of advanced communication features were added including messaging, calling, and contact management. The collaboration experience was enhanced with webcams, video playback, white-boarding, security features and more.
